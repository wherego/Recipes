# 菜谱大全

![icon](https://github.com/i7guokui/Recipes/blob/master/app/src/main/res/mipmap-xxhdpi/ic_launcher.png)

##介绍

菜谱大全是一个基于Android平台的菜谱应用，数据来源于网上寻找的免费API。
主要用到的开源库：RxJava、Retrofit2、Butterknife、Glide等
